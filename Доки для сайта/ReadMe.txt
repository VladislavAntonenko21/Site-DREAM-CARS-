Для запуска веб-сайта "DREAM CARS" на локальном ПК требуется:
- скачать и установить PyCharm,
- скачать и установить Python 3.9,
- открыть проект в PyCharm, далее в консоли прописать команду для запуска сервера: python manage.py runserver,
- далее открыть браузер и перейти по ссылке выданной в консоли PyCharm.
